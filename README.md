# Log Catcher

Log Catcher は、Webページ上で選択した文章をその場でカテゴリ付きメモとして保存できる、軽量な Chrome 拡張です。

閲覧中に見つけた気になる文章や、あとで見返したい情報を、ブラウザ作業の流れを止めずに素早く記録できるようにすることを目的に作成しました。

---

## 概要

Web を見ていると、ふとした瞬間に残しておきたい情報が出てきます。

- あとで買いたいもの
- メモしておきたい金額
- そのまま残したい文章
- TODO にしたい内容

Log Catcher は、そうした断片を「選択 → 右クリック → 保存」という短い流れで記録できる拡張です。

保存した内容はローカルに保持され、拡張のポップアップ画面から一覧・完了管理・削除・JSONでの入出力ができます。

---

## 主な機能

- Webページ上で選択した文章をそのまま保存
- 右クリックメニューからカテゴリを選んで登録
- 以下のカテゴリに対応
  - 金額
  - TODO
  - 買うもの
  - メモ
- ページ上に軽量な入力モーダルを表示
- タイトル、メモ、金額を補足入力可能
- `chrome.storage.local` を使ったローカル保存
- ポップアップ画面で一覧表示
- カテゴリごとの絞り込み表示
- 完了 / 未完了の切り替え
- 個別削除
- 全件クリア
- JSON形式でのエクスポート / インポート

---

## 想定している使い方

- 調べもの中に見つけた気になる商品を残す
- 記事やドキュメントを読みながら TODO を拾う
- サービス料金や価格をメモして比較材料にする
- その場で気になった文章を簡易メモとして保存する
- ブラウザ内で完結する軽量な個人ログとして使う

---

## 動作の流れ

1. Webページ上の文章を選択
2. 右クリックして Log Catcher のメニューを選ぶ
3. 必要に応じてタイトルやメモを補足
4. 保存
5. ポップアップから一覧を確認・管理

できるだけ操作の流れを止めず、すぐ残せることを重視しています。

---

## この拡張を作った理由

大きなノートアプリやタスク管理ツールを開くほどではないけれど、  
「この一文だけ残したい」  
「これだけ今メモしたい」  
という場面は日常的にあります。

Log Catcher は、そうした小さな記録をもっと気軽に扱うための拡張として作成しました。

目的は高機能な情報管理システムを作ることではなく、  
**日常のブラウザ作業の中にある、小さな不便を軽くすること**  
です。

## 技術構成

- JavaScript
- Chrome Extensions API (Manifest V3)
- Service Worker
- Content Scripts
- HTML / CSS
- `chrome.storage.local`
- `chrome.downloads`

---

## ファイル構成

### `manifest.json`
拡張全体の設定、権限、エントリーポイントを定義

### `service_worker.js`
右クリックメニューの作成、保存処理、JSONの入出力などのバックグラウンド処理を担当

### `content.js`
ページ上でのモーダル表示、入力受付、保存フローを担当

### `popup.html`
拡張ポップアップのUI構造

### `popup.js`
一覧表示、絞り込み、完了切り替え、削除、インポート / エクスポート操作を担当

### `popup.css`
ポップアップ画面のスタイル定義

---

## インストール方法

1. このリポジトリをダウンロードまたはクローン
2. `chrome://extensions/` を開く
3. 右上の **デベロッパーモード** を有効化
4. **パッケージ化されていない拡張機能を読み込む** をクリック
5. このプロジェクトのフォルダを選択
